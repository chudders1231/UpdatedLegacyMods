MoreQuickSlots
    by RandyKnapp

// Standalone Injector Instructions:
//////////////////////////////////////////////////////////////////////////
1. Extract MoreQuickSlots_Standalone.zip to some location
2. Copy the contents of Managed to <SubnauticaInstallDirectory>\Subnautica_Data\Managed\
3. Run ModInjector_MoreQuickSlots.exe
4. Make sure the directory in the "Subnautica Directory" field is correct, use the folder button to select a new directory if necessary
5. Click "Inject"
6. You should get a message saying that the mod was installed successfully
7. Run the game

// QMods Instructions:
//////////////////////////////////////////////////////////////////////////
1. Extract MoreQuickSlots_Standalone.zip to some location
2. Copy the contents of QMods to <SubnauticaInstallDirectory>\QMods\
3. Copy 0Harmony.dll to <SubnauticaInstallDirectory>\Subnautica_Data\Managed\
4. Run the game

// Config
//////////////////////////////////////////////////////////////////////////
1. Open mod.json in a text editor
2. Under the "config" object, edit the values as you want
  - SlotCount: [Number between 1 and 12] Sets the number of quick slots
  - ShowInputText: [Bool] Show or hide the input labels on the quick slots
